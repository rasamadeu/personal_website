CKM unitarity puzzle with Abelian flavour symmetries

The Standard Model (SM) of particle physics is considered to be one of the greatest theoretical achievements of Physics. It successfully describes three [electroweak (EW) and strong] out of the four fundamental interactions among the elementary particles known to mankind. However, we today know that it is an incomplete theory! For example, neutrino oscillations, which indicate that neutrinos are massive particles and that there is lepton mixing, is a phenomena that requires beyond the SM (BSM) physics in order to be accounted for. As a result, any experimental evidence or hint which contradicts the SM is crucial, since these help physicists to better understand the Universe.

In my thesis, I adressed a recent development in the measurement of the first row of the Cabibbo-Kobayashi-Maskawa matrix, which is in tension with the prediction of unitary quark mixing, commonly known as Cabibbo angle anomaly. The proposed model is a SM extension with a vector-like quark (VLQ). Furthermore, to increase the predictivity of the theory, the number of free parameters in the Yukawa and mass matrices is reduced by imposing texture zeros using Abelian symmetries, while maintaining compatibility with data. To achieve this, the SM must be further extended with a second Higgs doublet. For more details, you can check my thesis here.

During this project, I autonomously learned a lot about Python, specifically the Numpy and iMinuit packages, having developed several modules to obtain the major results in the thesis. The plots were made with matplotlib. The code is on my Github page.

Yes, I know it is a bit "meta" to refer to my personal webpage as a project in my personal webpage... Gosh, let me stop saying personal webpage, it's getting a bit cringy.

While watching the CS50's lecture on HTML, CSS and Javascript, I decided to challenge myself to create this website. The goal was to develop a basic understanding of these web development building blocks and to write everything myself.

Simulation of two strings attached to a pendulum

This projects consists of a GUI which allows the user to control several parameters of the simulation.

It consists of 3 panels:

  • Plots: displays two real-time plots of the motion. You can choose from a variety of coordinates to display.
  • Simulation: displays the motion of the masses and allows the user to control the speed of simulation and collision with wall.
  • System parameters and Initial conditions: allows the user to control the physical parameters of the masses and springs and set the initial conditions of the motion.

This program was written in C, and the GUI was built using the GTK+ library. You can check the code here.
